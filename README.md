# Car Sharing SPA (Second Iteration)

## Used libraries:

- Typescript 4.5+
- React 17+
- Bootstrap 5.1.3
- @reduxjs/toolkit
- react-router 6+
- react-helmet
- classnames
- gh-pages

## Commands

Run application in dev mode:

```shell
npm start
```

Build production bundle:

```shell
npm run build
```
